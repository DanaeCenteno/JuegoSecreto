# JS Game 🎮
Juego interactivo desarrollado con JavaScript, HTML y CSS donde el objetivo es adivinar un número secreto generado aleatoriamente. Al lograrlo, el juego te muestra cuántos intentos necesitaste para encontrarlo.

![](https://github.com/DanaeCenteno/JuegoSecreto/blob/main/JuegoSecreto.png)


**¿Cómo se juega?**

1. El juego genera un número secreto al azar
2. Ingresa tu intento en el campo de texto
3. El juego te indica si el número secreto es mayor o menor
4. Sigue intentando hasta adivinarlo
5. Al ganar, el juego te muestra el número de intentos que usaste

**Tecnologías utilizadas**

* HTML5
* CSS3
* JavaScript

**Aprendizajes clave

* Manipulación del DOM con JavaScript
* Lógica de control de flujo y condicionales
* Manejo de eventos del usuario
* Generación de números aleatorios con Math.random()
* Contador de intentos con variables de estado
